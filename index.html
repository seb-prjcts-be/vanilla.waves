one dialect, three runtimes

same numbers, no canvas

The p5.waves math never actually needed p5 — wave(), the 34 formulas, the seeded shift engine, all of it is pure Math.*. vanilla.waves is that canonical core, byte-for-byte, with the p5 hook removed and a tiny DOM engine bolted on.

drop it in

One tag pulls the math plus the DOM engine. Mark an element with data-wv; one shared 30 fps loop animates every marked element and pauses the ones offscreen.

<!-- one tag: math + DOM engine -->
<script src="https://cdn.jsdelivr.net/gh/seb-prjcts-be/vanilla.waves@v0.1.0/vanilla.waves.min.js"></script>
<div data-wv="eq" data-seed="13"></div>
canvasless demos

what one loop drives

The sampler returns a number; you map it to any DOM property. Height, position, a character, an opacity — all from the same breathing, shifting, morphing motion.

equalizer

sampler → scaleY

Each bar's height is one sampler value read at a different position. The wave shifts through formulas on its own, so the meter never repeats.

// each bar: a sampler value → scaleY
v = 0.08 + s.sample(i * 0.55, t) * 0.92;
bar.style.transform = 'scaleY(' + v + ')';

wave line

sampler → translateY

One sampler traced across position, drawn as a row of dots. The row is the live waveform, complete with shift and morph.

// dots trace the waveform
v = s.sample(i * 0.18, t);
dot.style.transform = 'translateY(' + v*38 + 'px)';

text loader

wave → character

A 4×4 grid where each cell picks a glyph from a wave value. Cheap enough to run dozens at once — the canvasless loader technique.

// value → index into a character ramp
idx = floor((v + 1) / 2 * text.length);
cell.textContent = text[idx];

tone field

wave → opacity

A grid of ink cells whose opacity is a sampler value. Structured like noise, but it breathes and re-forms as the wave shifts.

// each cell: value → opacity
v = (s.sample(c*0.4, t + r*0.5) + 1) / 2;
cell.style.opacity = 0.06 + v * 0.94;

just the math

drive any property

Skip the engine entirely if you want. A sampler is one value in, one number out — point it at any DOM property in your own loop.

// same numbers as p5.waves — drive anything
const s = VanillaWaves.createSampler({ shift: true, range: [-50, 50] });

function tick(now) {
  el.style.transform = 'translateY(' + s.sample(0, now/1000) + 'px)';
  requestAnimationFrame(tick);
}
requestAnimationFrame(tick);
