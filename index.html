Equalizer — the DOM engine

A registered eq type. Each bar's height is a sampler value; one shared 30 fps loop drives them all.

Wave line — a sampler traced in dots

One sampler, sampled across position; the row of dots traces the live waveform as it shifts and morphs.

Text loader — characters from wave density

The canvasless loader technique: a 4×4 grid where each cell picks a character from a wave value. Cheap enough to run dozens at once.
