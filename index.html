p5.waves drives a canvas; vanilla.waves drives real elements. Each block below is an ordinary <input>, <progress>, checkbox grid or text node — one shared 30 fps loop samples a wave and writes it into the DOM.

Just the math

The wave math is the same byte-identical port — 1009/1009 parity checks against p5.waves v3.4.0. One number out; drive any property you like.

// same numbers as p5.waves — drive any DOM property const s = VanillaWaves.createSampler({ shift: true, range: [-50, 50] }); function frame(t) { el.style.transform = 'translateY(' + s.sample(0, t) + 'px)'; requestAnimationFrame(frame); }
One script tag. No build step.

Get started

Add the tag. Drop a data-wv element. The engine auto-wires it.
