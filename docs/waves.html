Periodicity

A wave is periodic when there is a distance T such that wave(x) == wave(x + T) for every x. That distance is the wave's period. Knowing it lets you close a shape cleanly: sample a wave around a ring over an integer number of periods and the last point lands exactly on the first — no seam.

Why 62.83 and not 360? wave(x) treats x as a generic coordinate — the same mental model as noise(x) — not an angle. Ten of the 34 waves are not sine-based at all (square, triangle, ramp, noise…), so radians would be meaningless for them.

Every period value was measured, not derived: the periodicity harness in the p5.waves repo samples each wave, finds the smallest T where the mean error between wave(x) and wave(x + T) stays below 0.002, then confirms with multiples. vanilla.waves produces numerically identical output, so the same periods hold.

Closed-ring recipe (canvasless)

No <canvas>, no beginShape — build an SVG path. The integer lobes is the only thing you tune. Read the sweep straight off the sampler with .period so a shifting ring never tears its seam (see the guide).

// A closed wobble ring in SVG. No visible seam.
const ring = VanillaWaves.createSampler({
  group: 'closing', shift: true, amplitude: 30
});
const lobes = 8, steps = 240;
const sweep = ring.period * lobes;   // integer multiple = clean close

let d = '';
for (let i = 0; i < steps; i++) {
  const a = (i / steps) * Math.PI * 2;
  const r = 140 + ring.sample(i / steps * sweep, t);
  d += (i ? 'L' : 'M') + (200 + r*Math.cos(a)) + ' ' + (200 + r*Math.sin(a));
}
path.setAttribute('d', d + 'Z');   // Z closes it

How to read the result

  • Periodic — a number. Use any integer multiple of it as your sweep.
  • — — quasi-periodic (incommensurate rhythms) or non-periodic (growing amplitude, Perlin noise, random branches). Close with a soft blend back to the starting value instead.

Want a shape that stays closed while shifting between formulas? Use group: 'closing' — a pool of 17 waves that all share the base period 62.8319 — and read the sweep from sampler.period. See Closing shapes in the guide.
