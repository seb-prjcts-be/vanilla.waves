One Number

One call returns one number. Use it for size, position, colour — anything. Here it drives three real elements at once.

one sampler, three properties—
// one number, used three ways
const s = VanillaWaves.createSampler({ shift: true, range: [0, 1] });

button.style.minWidth = (3 + s.sample(0, t) * 7) + 'rem';   // size
mover.style.transform = 'translateX(' + (s.sample(2, t)-.5)*180 + 'px)'; // position
const g = 20 + s.sample(4, t) * 200;
dot.style.background = `rgb(${g},${g},${g})`;                // colour

Bar Meter

p5.waves' shifting ribbons, as a real equalizer. Each <span> bar is one sample; the formula morphs every few seconds.

24 bars · one sampler—
const s = VanillaWaves.createSampler({ shift: true, range: [0, 1] });

// each bar samples the same wave at its own x
for (let i = 0; i < bars.length; i++) {
  const v = 0.06 + s.sample(i * 0.5, t) * 0.94;
  bars[i].style.transform = `scaleY(${v})`;
}

Slider Stack

Native <input type="range"> elements whose value is the wave. The thumbs glide, the readouts tick — no custom widget, just real form controls.

6 native sliders—
const s = VanillaWaves.createSampler({
  shift: true, range: [0, 100], frequency: 0.35
});

for (let i = 0; i < rows.length; i++) {
  const v = Math.round(s.sample(i * 0.9, t + i * 0.15));
  rows[i].range.value = v;        // a real form control
  rows[i].output.textContent = v;
}

Checkbox Field

The binary field, on a grid of real checkboxes. One sampler per axis; sum them per cell; threshold the result. The whole 2D pattern toggles itself.

14 × 7 checkboxes—
const rowS = VanillaWaves.createSampler({ shift: true, range: [-1, 1], seed: 1 });
const colS = VanillaWaves.createSampler({ shift: true, range: [-1, 1], seed: 2 });

for (let r = 0; r < rows; r++) {
  const rv = rowS.sample(r * 5 + t * 0.4, t);
  for (let c = 0; c < cols; c++) {
    box[r][c].checked = (rv + colS.sample(c * 2.5 - t * 0.4, t)) > 0;
  }
}

Progress Morph

Rows of native <progress>, each on its own shift-sampler and frequency. They fill, drain and re-fill as their formulas change — the label shows the live wave name.

5 native progress barsindependent shift
// one sampler per bar, each its own seed + frequency
rows.forEach((r, i) => {
  r.sampler = VanillaWaves.createSampler({
    shift: true, range: [0, 100], seed: i * 31 + 5, frequency: 0.3 + i * 0.05
  });
});

rows.forEach((r, i) => {
  r.progress.value = r.sampler.sample(i * 1.3, t);
  r.label.textContent = r.sampler.waveName;   // live formula name
});

Glyph Field

A grid of / - | \ characters in a single text node. One sampler picks the direction per cell — like noise(), but with unexpected wave shifts every few seconds.

30 × 12 glyphs—
const DIRS = ['-', '/', '|', '\\'];
const s = VanillaWaves.createSampler({ shift: true, frequency: 2 });

let out = '';
for (let r = 0; r < ROWS; r++) {
  for (let c = 0; c < COLS; c++) {
    const v = s.sample(c * 0.5, t + r * 0.4);       // ~[-1, 1]
    const i = Math.floor((v + 1) / 2.002 * DIRS.length);
    out += DIRS[i];
  }
  out += '\n';
}
field.textContent = out;     // one text node, no canvas

Radial Loader

Dots placed around a ring with transform: rotate() translateY(). The closing pool shares one base period, so a sweep of period × lobes closes through every morph — the loop never shows a seam.

56 dots · closing pool—
const s = VanillaWaves.createSampler({ shift: true, group: 'closing', amplitude: 1 });
const LOBES = 6;

// s.period comes from the library — 6.2832 for the closing pool
const sweep = s.period * LOBES;
for (let i = 0; i < N; i++) {
  const frac = i / N;
  const r = 62 + s.sample(frac * sweep, t) * 26;
  dots[i].style.transform = `rotate(${frac*360}deg) translateY(-${r}px)`;
}

Chip Walkers

No angles, no steps — wave output is the velocity. Small badges drift across the field, each with its own pair of samplers, wrapping at the edges.

9 walking chipsvelocity = wave
const xs = VanillaWaves.createSampler({ shift: true, range: [-1.4, 1.4], seed: 3 });
const ys = VanillaWaves.createSampler({ shift: true, range: [-1.4, 1.4], seed: 9 });

// wave output IS the velocity — accumulate, then wrap
chip.x += xs.sample(t * 1.8, t);
chip.y += ys.sample(t * 2.1, t);
chip.el.style.transform = `translate(${chip.x}px, ${chip.y}px)`;

Text Loader

The canvasless loader: a 4×4 grid where each cell picks a character from a wave value. Cheap enough to run dozens at once — swap the ramp to change the texture.

three ramps, three seeds4×4
const s = VanillaWaves.createSampler({ shift: true });
const ramp = '·:░▒▓█';

for (let r = 0; r < 4; r++)
  for (let c = 0; c < 4; c++) {
    const v = s.sample(c * 0.5 + t, t + r * 0.9);     // ~[-1, 1]
